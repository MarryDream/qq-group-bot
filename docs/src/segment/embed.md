# Embed消息段

- 可单独发送
- 可与 [回复(ReplyElem)](./reply.md)一起发送
- 仅频道和频道私信支持
- 发送格式

| 属性名       | 属性值             |
|-----------|-----------------|
| type      | button          |
| title     | string          |
| prompt    | string          |
| thumbnail | {url:string}    |
| fields    | {name:string}[] |
