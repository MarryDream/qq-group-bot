# Markdown消息段

- 可与 [回复(ReplyElem)](./reply.md)一起发送
- 可与多个 [按钮(ButtonElem)](./button.md) 发送
- 发送格式

| 属性名                | 属性值                         |
|--------------------|-----------------------------|
| type               | markdown                    |
| content            | string                      |
| custom_template_id | string                      |
| params             | {key:string,value:string}[] |
